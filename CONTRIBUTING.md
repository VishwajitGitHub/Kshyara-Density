# Contributing to KSHYARA CLI

We welcome contributions! Please follow these steps:

1. Fork the repository.
2. Create a new branch for your feature (`git checkout -b feature/amazing-feature`).
3. Commit your changes (`git commit -m 'Add amazing feature'`).
4. Push to the branch (`git push origin feature/amazing-feature`).
5. Open a Pull Request.

## Architecture
- `cli/`: Core CLI interface, UI, and commands.
- `core/`: Advanced engines (Combiner, Debate, Orchestrator).
- `agents/`: Specialized AI agents (Planner, Coder, Critic).
- `tools/`: Extensible tool system (Web, Code, FS).
