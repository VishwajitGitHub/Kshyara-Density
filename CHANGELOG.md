# Changelog

All notable changes to KSHYARA CLI will be documented in this file.

## [1.0.0] - Initial Release
### Added
- Core CLI interactive loop with `readline`.
- Multi-model routing and mock AI streaming layer.
- Combiner Engine for merging multiple model outputs.
- Debate Engine for multi-round AI arguments.
- 19 Slash commands including `/ask`, `/combine`, `/debate`, `/config`, `/theme`.
- Global `kshyara` command registration.
